## Tentang Proyek
**love.exe** adalah program sederhana yang dirancang khusus untuk menyampaikan pesan-pesan manis dan romantis kepada orang tersayang. Proyek ini cocok untuk para pacar bucin yang ingin mengekspresikan cinta dengan cara yang unik dan menggemaskan. Program ini dapat memberikan inspirasi, motivasi, atau sekadar pesan harian yang membuat hari lebih berwarna.

## Fitur
- **Pesan Cinta Acak:** Setiap kali dijalankan, love.exe akan menampilkan pesan cinta yang berbeda.
- **Antarmuka Minimalis:** Tampilan yang simpel namun elegan untuk memberikan kesan romantis.
- **Personalisasi Pesan:** Mudah untuk menambah, mengubah, atau menghapus pesan sesuai selera.
- **Portabilitas:** File executable (.exe) yang dapat dijalankan di berbagai sistem Windows tanpa perlu instalasi tambahan.
